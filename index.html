Sobre o projeto

Um sistema para acompanhar a vida com mais clareza.

O Fluir foi desenvolvido como um sistema de organização pessoal. A proposta é centralizar recursos do dia a dia em uma interface limpa, com versão web e adaptação para dispositivos móveis.

Funcionalidades

O que o sistema oferece

Tarefas

Criação e acompanhamento de atividades importantes da rotina.

Hábitos

Controle de hábitos pessoais com foco em constância e progresso.

Hidratação

Registro de consumo de água e acompanhamento diário.

Finanças

Organização de entradas, saídas e metas financeiras.

Sono

Acompanhamento da rotina de descanso e qualidade do sono.

Perfil

Área de usuário com informações pessoais e preferências.

Versões do sistema

O Fluir foi pensado para funcionar em diferentes plataformas.

Sistema Web

Versão acessível pelo navegador, ideal para uso em computadores, notebooks, tablets e celulares, com acesso direto pela internet.

App Android

Aplicativo mobile planejado para celulares Android, com interface própria para uso diário no celular.

App iOS

Versão futura para iPhone, mantendo a mesma proposta do Fluir em um aplicativo adaptado ao sistema iOS.

Desktop Futuro

Versão planejada para computadores, com foco em uma experiência mais completa para uso em tela grande.
